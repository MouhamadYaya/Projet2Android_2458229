## Description de l'application
Bibliothèque est une application Android qui me permet de gérer ma collection personnelle de livres.
Avec cette application, je peux :
Voir tous mes livres : une liste avec les images ,les titres et l'auteur
Ajouter un livre : remplir le titre, l’auteur et le type
Chercher un livre : utiliser le filtre pour trouver rapidement un livre dans ma collection.
Voir les détails d’un livre : quand je clique sur un livre, je peux voir toutes ses informations.
L’application a un thème clair et sombre que je peux changer facilement, et elle fonctionne en français, anglais et espagnol.
Tous mes livres sont sauvegardés dans l’application, donc je ne perds rien entre les sessions.
